qT3: mobile app

qT3 is a prototype iOS app which algorithmically deduces information from Arabic verbs, including form, tense, root, person, gender and number. It is intended to help beginnning students of Arabic get more information about a verb than they can from any existing dictionary lookups.

For more information checkout the GitHub project, linked here.
